## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** 18 or later
- **npm** or **pnpm**
- **Git**

## Installation

```bash
# Clone the repository
git clone https://github.com/twineline/twineline.git
cd twineline

# Install dependencies
npm install

# Start the development server
npm run dev
```
